<Info>You need to use your API key to connect Zapier to your Inboundly workspace.</Info>

To get started, go to the Settings tab, and select "Integrations" from the sidebar.

<video autoplay="true" muted="true" loop playsinline className="w-full aspect-video" src="/videos/guides_zapierinvite.mp4" />

Under Platforms, click "Get Started" and accept the Zapier invite.

Then, create a new Zap and search for "Inboundly". Then, continue creating your Zap as needed.

You can choose from several triggers on Zapier. Here's what each of them means:
<AccordionGroup>
	<Accordion title="New Lead Created">Triggers when a new lead is created. Each lead object contains their  X/Twitter name and handle, the inbox they are associated with, and the date they were created.</Accordion>
	<Accordion title="New List Created">Triggers when a new list is created. Each list object contains the list's name, tag, and the inbox they are associated with.</Accordion>
	<Accordion title="Lead Added to List">Triggers when a lead is added to a list. Each lead object contains their  X/Twitter name and handle, the inbox they are associated with, and the name and tag of the list they were added to.</Accordion>
</AccordionGroup>
